    以下の最適化問題を考える

    min 12xTQx+cTxs.t. Ax=b,Aeqx≤beq,lx≤x≤uxx∈Rn \text{min} \space \dfrac{1}{2}x^TQx + c^T x\\ \text{s.t.} \space Ax = b, A_{eq}x \leq b_{eq}, l_x \leq x \leq u_x \\ x \in \R^n min 21​xTQx+cTxs.t. Ax=b,Aeq​x≤beq​,lx​≤x≤ux​x∈Rn

    探索方向の決定

    KKT条件に対して各変数を微小変化させた場合を考える

    x←x+Δxs←s+Δsλ←λ+Δλλeq←λeq+Δλeq x \leftarrow x + \Delta x\\ s \leftarrow s + \Delta s\\ \lambda \leftarrow \lambda + \Delta \lambda\\ \lambda_{eq} \leftarrow \lambda_{eq} + \Delta \lambda_{eq}\\ x←x+Δxs←s+Δsλ←λ+Δλλeq​←λeq​+Δλeq​
