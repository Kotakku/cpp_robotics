    多項式補間

    ロボットアームなどをなめらかに動かしたい時に目標値をなめらかに動かすことで実現しようとした時になるべく簡単に参照軌道を作りたいことがある。簡単に実装するなら多項式で軌道を作るのが簡単である。

    1次式

    境界条件数2

    始点位置P0P_0P0​, 終点位置P1P_1P1​, 間隔TTTのとき

    f(t)=a0+a1t f(t) = a_0 + a_1 t f(t)=a0​+a1​t

    a0=P0a1=P1−P0T \begin{array}{rcl} a_0 &=& P_0\\ a_1 &=& \dfrac{P_1 - P_0}{T} \end{array} a0​a1​​==​P0​TP1​−P0​​​

    2次式

    境界条件数3

    始点(P0,v0)(P_0, v_0)(P0​,v0​), 終点位置(P1)(P_1)(P1​), 間隔TTTのとき

    f(t)=a0+a1t+a2t2 f(t) = a_0 + a_1 t + a_2 t^2 f(t)=a0​+a1​t+a2​t2

    a0=P0a1=v0a2=1T2{(P1−P0)−v0T} \begin{array}{rcl} a_0 &=& P_0\\ a_1 &=& v_0\\ a_2 &=& \dfrac{1}{T^2}\left\{ (P_1 - P_0) - v_0T \right\}\\ \end{array} a0​a1​a2​​===​P0​v0​T21​{(P1​−P0​)−v0​T}​

    始点(P0,v0)(P_0, v_0)(P0​,v0​), 終点位置(v1)(v_1)(v1​), 間隔TTTのとき

    f(t)=a0+a1t+a2t2 f(t) = a_0 + a_1 t + a_2 t^2 f(t)=a0​+a1​t+a2​t2

    a0=P0a1=v0a2=12T(v1−v0) \begin{array}{rcl} a_0 &=& P_0\\ a_1 &=& v_0\\ a_2 &=& \dfrac{1}{2T}(v_1 - v_0)\\ \end{array} a0​a1​a2​​===​P0​v0​2T1​(v1​−v0​)​

    3次式

    境界条件数4

    始点(P0,v0)(P_0, v_0)(P0​,v0​), 終点位置(P1,v1)(P_1, v_1)(P1​,v1​), 間隔$Tx$のとき

    f(t)=a0+a1t+a2t2+a3t3 f(t) = a_0 + a_1 t + a_2 t^2 + a_3 t^3 f(t)=a0​+a1​t+a2​t2+a3​t3

    a0=P0a1=v0a2=1T2{3(P1−P0)−(v1+2v0)T}a3=1T3{−2(P1−P0)−(v1+v0)T} \begin{array}{rcl} a_0 &=& P_0\\ a_1 &=& v_0\\ a_2 &=& \dfrac{1}{T^2}\left\{ 3(P_1 - P_0) - (v_1 + 2v_0)T \right\}\\ a_3 &=& \dfrac{1}{T^3}\left\{ -2(P_1 - P_0) - (v_1 + v_0)T \right\} \end{array} a0​a1​a2​a3​​====​P0​v0​T21​{3(P1​−P0​)−(v1​+2v0​)T}T31​{−2(P1​−P0​)−(v1​+v0​)T}​

    4次式

    境界条件数5

    始点(P0,v0,acc0)(P_0, v_0, acc_0)(P0​,v0​,acc0​), 終点位置(v1,acc1)(v_1, acc_1)(v1​,acc1​), 間隔TTTのとき

    f(t)=a0+a1t+a2t2+a3t3+a4t4 f(t) = a_0 + a_1 t + a_2 t^2 + a_3 t^3 + a_4 t^4 f(t)=a0​+a1​t+a2​t2+a3​t3+a4​t4

    a0=P0a1=v0a2=12acc0a3=13T2{3(v1−v0)−(acc1+2acc0)T}a4=14T3{−2(v1−v0)+(acc1+acc0)T} \begin{array}{rcl} a_0 &=& P_0\\ a_1 &=& v_0\\ a_2 &=& \dfrac{1}{2} acc_0\\ a_3 &=& \dfrac{1}{3T^2}\left\{ 3(v_1 - v_0) - (acc_1 + 2acc_0)T \right\}\\ a_4 &=& \dfrac{1}{4T^3}\left\{ -2(v_1 - v_0) + (acc_1 + acc_0)T \right\}\\ \end{array} a0​a1​a2​a3​a4​​=====​P0​v0​21​acc0​3T21​{3(v1​−v0​)−(acc1​+2acc0​)T}4T31​{−2(v1​−v0​)+(acc1​+acc0​)T}​

    境界条件数6

    始点(P0,v0,acc0)(P_0, v_0, acc_0)(P0​,v0​,acc0​), 終点位置(P1,v1,acc1)(P_1, v_1, acc_1)(P1​,v1​,acc1​), 間隔TTTのとき

    f(t)=a0+a1t+a2t2+a3t3+a4t4+a5t5 f(t) = a_0 + a_1 t + a_2 t^2 + a_3 t^3 + a_4 t^4 + a_5 t^5 f(t)=a0​+a1​t+a2​t2+a3​t3+a4​t4+a5​t5
